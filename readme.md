## Quick Start
1. Open `index.html` in a web browser
2. For mobile testing, use Chrome DevTools device emulator
3. For haptic testing, need actual Android device

## Project Structure
- `index.html` - Main product page
- `cart.html` - Shopping cart
- `css/style.css` - All styling
- `js/products.js` - Product data
- `js/cart.js` - Cart functionality
- `js/haptics.js` - Vibration API wrapper
